# Day 1 - Exploratory Data Analysis and Preprocessing

## Day 1 Goal

Day 1 builds the foundation needed before machine learning algorithms.

The focus is not advanced prediction yet. The focus is understanding data:

- What data looks like
- How Python handles data
- How to inspect data
- How to clean data
- How to summarize data
- How to visualize data
- How to prepare data for later modeling

## Day 1 Scope

Included:

- Data science and AI/ML ecosystem
- Python library orientation
- Notebook usage
- Environment verification
- First simple examples with Python objects, NumPy, and Pandas
- Python variables, lists, and dictionaries
- NumPy arrays and basic calculations
- Pandas Series and DataFrames
- Selecting, filtering, sorting, adding columns, and grouping
- Data inspection with shape, columns, head, and info
- Missing value detection
- Duplicate detection
- Text cleaning
- Missing value handling
- Saving raw and cleaned datasets
- Mean, median, and mode
- Variance and standard deviation
- Range and percentiles
- Covariance and correlation
- Distribution and skewness
- Chart selection based on questions
- Bar charts, line charts, histograms, box plots, scatter plots, and count plots
- Chart formatting and plain-English observations
- Numeric scaling
- Min-max normalization
- One-hot encoding
- Ordered category mapping
- Saving a preprocessed dataset
- Mini EDA project
- Final Day 1 EDA report
- Evidence-based observations and recommendations

Not included yet:

- Regression algorithms
- Classification algorithms
- Train-test split
- Model training
- PyCaret
- Neural networks
- Generative AI

Those topics come in later days.
