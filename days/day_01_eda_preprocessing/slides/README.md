## How to Use

Use `SECTION_01_AI_WORLD_AND_DATA_SCIENCE.md` as the main opening deck before Notebook 01.

Use `SECTION_02_DATA_SCIENCE_LIFECYCLE_AND_PROJECT_FLOW.md` after Section 1 and before starting Notebook 01.

Use `SECTION_03_DATA_INSPECTION_AND_CLEANING.md` before Notebook 03, after learners have completed the setup and Pandas foundations notebooks.

Use `SECTION_04_BASIC_STATISTICS_FOR_DATA_UNDERSTANDING.md` before Notebook 04, after learners have cleaned and saved the customer activity dataset.

Use `SECTION_05_DATA_VISUALIZATION_WITH_MATPLOTLIB_SEABORN.md` before Notebook 05, after learners understand basic statistics and are ready to communicate patterns visually.

Use `SECTION_06_SCALING_NORMALIZATION_ENCODING.md` before Notebook 06, after learners have practiced statistics and visualization and are ready to prepare columns for later workflows.

Suggested delivery:

1. Start with the AI world story.
2. Explain why AI became powerful now.
3. Show the AI family tree.
4. Clarify where Data Science fits.
5. Explain the Data Science lifecycle.
6. Run the setup demo using `../SETUP_DEMO_CHECKLIST.md`.
7. Open Notebook 01.
8. Use Section 3 before Notebook 03 to frame inspection and cleaning as trust-building work.
9. Use Section 4 before Notebook 04 to frame statistics as practical summary language.
10. Use Section 5 before Notebook 05 to frame charts as visual evidence, not decoration.
11. Use Section 6 before Notebook 06 to frame preprocessing as column preparation for later workflows.
