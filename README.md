# Applied Data Science and Machine Learning

Hands-on training program for learning Applied Data Science and Machine Learning step by step.

## Start Here

1. Complete the environment setup in `SETUP.md`.
2. Open the notebook and run it top to bottom.

## Day-Wise Learning Path

### Day 1: Data Science Lifecycle Demo

One notebook covering the full Data Science lifecycle:

- `notebooks/day_01_data_science_lifecycle_demo.ipynb`

Lifecycle flow in the notebook:

1. Problem Understanding
2. Data Collection
3. Data Understanding
4. Data Cleaning
5. Exploratory Data Analysis
6. Statistics
7. Visualization
8. Feature Preparation
9. Communication and Decision

### Later Days

- Day 2: Predicting numbers from data
- Day 3: Predicting categories from data
